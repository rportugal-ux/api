# 📦 Loja Virtual API

## 📌 Nome do Projeto
Loja Virtual API

---

## 🎯 Objetivo da API
Esta API tem como objetivo gerenciar produtos de uma loja virtual, permitindo operações de cadastro, listagem, atualização e remoção de produtos.

## 🔗 Endpoints Desenvolvidos

### Produtos
- GET /produtos → Lista todos os produtos
- GET /produtos/:id → Busca produto por ID
- POST /produtos → Cadastra novo produto
- PUT /produtos/:id → Atualiza produto existente
- DELETE /produtos/:id → Remove produto
